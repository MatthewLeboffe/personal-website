my classes @ cmu

15-210 is designed to impart methods for designing, analyzing, and programming both sequential and parallel algorithms and data structures using Standard ML (SML) . The core emphasis is on ingraining fundamental concepts that span across diverse problem domains and are transferable across a broad array of programming languages and computer architectures.

Distinctively, rather than a conventional introduction to algorithms and data structures, this course champions parallel thinking—envisioning algorithms executing multiple tasks simultaneously, as opposed to a sequential approach. It builds upon knowledge acquired in 15-122 and 15-150, delving deeper into algorithmic aspects. The course encompasses a substantial programming component where students tackle real-world examples from domains such as engineering, scientific computing, graphics, data mining, and information retrieval (web search).

Functional Programming (FP) in this course revolves around computation as evaluation, blending theory with practice. Unlike imperative programming, FP emphasizes expressions based on applicative behavior, guaranteeing type-safe evaluations. Its no side-effects stance paves the way for mathematical reasoning and enhanced runtime analysis.

Key advantages of FP include:

  • Verification: Seamless alignment of mathematical reasoning and program structure.
  • Parallelism: Side-effect free evaluations enabling concurrent operations.
  • Abstraction: Holistic data-centric computations with clear demarcations between implementation and interface.

This course dives deep into imperative programming using a C-like language, focusing on crafting correct and efficient programs. Designed for those versed in basic programming, the course transforms high-level algorithm descriptions into robust imperative implementations, particularly emphasizing basic data structures and algorithms. The journey begins with a verifiable subset of C and seamlessly transitions to full-fledged C.

Key takeaways:

  • Mastering the art of deliberate programming for high-quality code.
  • Understanding performance assessment and addressing deployment constraints.
  • Grasping core Computer Science concepts like abstraction, correctness, and modularity.
  • Gaining hands-on experience with C0 and C, enriching your skills in imperative programming while exploring vital data structures and algorithms.
